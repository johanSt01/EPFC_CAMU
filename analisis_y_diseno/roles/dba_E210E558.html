    Main Description

    El DBA realiza las siguientes tareas:

    Comprender los requisitos del sistema
    Construir el modelado conceptual
    Construir el modelado lógico
    Documentación del diseño
    Staffing
    Skills

    El DBA tiene algunas habilidades como:


    Administración de bases de datos
    Optimización de rendimiento
    Mantenimiento de bases de datos
    Configuración de seguridad de bases de datos
    Diseño y modelado de bases de datos
    Conocimiento de lenguajes de consulta de bases de datos (SQL)
    Gestión de usuarios y permisos
    Capacidad de trabajar con grandes volúmenes de datos
    Key Considerations

    El DBA:

    Debe construir una DB de acuerdo a la arquitectura presentada
    Debe construir una DB segura
    Debe construir un diseño de DB claro
    Debe tener buen pensamiento lógico
    Debe entender la estructura del negocio
