Role: Diseñador UI/UX
El diseñador UI/UX crea los bocetos de las interfaces de usuario para presentársela al desarrollador de frontend.
Relationships
Primary Performs
  • Capturar el modelo mental del usuario
  • Creacion de los flujos de usuario
  • Diseñar los mockups
Modifies
  • Diagrama de flujo de interaccion
  • Documentacion modelo mental
  • Mockups
    Main Description

    El diseñador UI/UX se encarga de:

    Capturar el modelo mental del usuario
    Creación de los flujos de usuario
    Diseñar los mockups
    Staffing
    Skills

    El diseñador UI/UX tiene algunas habilidades como:

    Diseño de mockups intuitivos y atractivos
    Diseño centrado en el usuario
    Pensamiento creativo y diseño visual
    Conocimientos de estándares de diseño y patrones de interacción
    Manejo de herramientas de diseño
    Colaboración con equipos de desarrollo
    Análisis de métricas y datos de usuario
    Conocimientos básicos de desarrollo web o móvil
    Key Considerations

    El diseñador UI/UX:

    Debe tener buena creatividad
    Debe realizar mockups interactivos y fáciles de comprender
    Debe realizar mockups que sean posibles de programar
    Debe realizar mockups que concuerden con el proyecto
