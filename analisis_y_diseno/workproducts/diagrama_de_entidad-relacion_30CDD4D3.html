Outcome: Diagrama de entidad-relacion
Es una representación visual para describir las entidades, sus atributos y las relaciones entre ellas
Purpose
capturar y representar la estructura lógica de la base de datos

Main Description
En el diagrama de entidad-relación, las entidades se representan mediante rectángulos, y los atributos se muestran dentro de esos rectángulos. Las relaciones entre las entidades se representan mediante líneas que conectan los rectángulos correspondientes
