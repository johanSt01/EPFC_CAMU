Role: DBA
El DBA se encarga de diseñar y documentar la estructura de la base de datos
Synonyms:
  • Administrador de bases de datos
  • Especialista en bases de datos
  • Gestor de bases de datos
  • Ingeniero de bases de datos
Relationships
Primary Performs
  • Ajustar rendimiento de la base de datos
  • Crear documentacion de la base de datos
  • Ejecucion de scripts en la base de datos
  • Gestion de los roles en la base de datos
  • Implementar modelo de datos
Modifies
  • DB optimizada
  • Diseno de la base de datos
  • Documentacion
  • Usuarios
    Main Description

    El DBA realiza las siguientes tareas:

    • Comprender los requisitos del sistema
    • Construir el modelado conceptual
    • Construir el modelado lógico
    • Documentación del diseño
    Staffing
    Skills

    El DBA tiene algunas habilidades como:

    • Administración de bases de datos
    • Optimización de rendimiento
    • Mantenimiento de bases de datos
    • Configuración de seguridad de bases de datos
    • Diseño y modelado de bases de datos
    • Conocimiento de lenguajes de consulta de bases de datos (SQL)
    • Gestión de usuarios y permisos
    • Capacidad de trabajar con grandes volúmenes de datos
    Assignment Approaches

    El DBA tiene diferentes enfoques de asignación tales como:

    • Realizar cualquier cambio según se solicite
    • Presentar el producto final al arquitecto de software
    • Documentar el desarrollo de la DB
    • Plantear una buena base de datos que cumpla con los requisitos
    Synonyms
    • Administrador de bases de datos
    • Especialista en bases de datos
    • Gestor de bases de datos
    • Ingeniero de bases de datos
    Key Considerations

    El DBA:

    • Debe construir una DB de acuerdo a la arquitectura presentada
    • Debe construir una DB segura
    • Debe construir un diseño de DB claro
    • Debe tener buen pensamiento lógico
    • Debe entender la estructura del negocio
