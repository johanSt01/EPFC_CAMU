Outcome: Informe de avance
El informe de avance va a contener todos los cambios que se han realizado en una pequeña parte del proyecto.
Purpose
El propósito de este avance es revisar como va el proyecto en pequeños avances para analizarlo y tomar la decisón correcta.

Main Description

En este avance de proyecto se tiene en cuenta una disciplina ya que debemos revisar como se ejecuto y si algo fallo o no para continuar con la siguiente por lo tanto debemos tener varios factores a revisar, estos son:

  • El tiempo que se tomo en hacer las actividades: Debemos tener en cuenta los tiempos de cada una de las actividades que se plantearon en el plan de proyecto
  • Actividad correcta: debemos revisar que la actividad que se realizo si este correcta respecto al plan de proyecto
  • Analizar cada actividad que se realizó: revisar cada una de las actividades es importante para detectar posibles fallas que afectan la producción del futuro
