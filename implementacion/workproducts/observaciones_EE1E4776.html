Artifact: Observaciones
Las observaciones del proyecto son una pequeña información que contiene el avance del proyecto para informar a la persona interesada sobre los avances.
Purpose
Las observaciones tienen como propósito informar al lider del proyecto sobre el avance y revisar si todo esta en orden.

Description
Main Description

Las observaciones puede contener varios aspectos a revisar tales como:

  • Desviaciones en el cronograma: Se observan retrasos en las actividades o hitos del proyecto en comparación con el cronograma planificado. Se pueden identificar las causas de los retrasos y se sugieren medidas correctivas para recuperar el tiempo perdido.
  • Desviaciones en el presupuesto: Se observan gastos o costos que exceden el presupuesto establecido. Se analizan las razones de las desviaciones y se proponen acciones correctivas para controlar los costos y mantener el proyecto dentro de los límites financieros.
  • Problemas de calidad: Se identifican problemas relacionados con la calidad de los entregables del proyecto. Se hacen observaciones sobre posibles deficiencias o no conformidades y se sugieren acciones correctivas para mejorar la calidad.
  • Riesgos no mitigados: Se observan riesgos que no han sido identificados o mitigados adecuadamente en el proyecto. Se recomienda revisar y actualizar la gestión de riesgos, implementando medidas para reducir o eliminar los impactos negativos.
