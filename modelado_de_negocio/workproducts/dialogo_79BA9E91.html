Artifact: Dialogo
El dialogo se da para que el gerente de a entender de que se trata el proximo proyecto a realizar
Purpose

Dar a conocer la empresa, lo que se quiere hacer, es decir, el proyecto que se va a ejecutar y el equipo de trabajo que lo va a hacer

Relationships
RolesResponsible:

TasksInput To:
  • Asignar equipo de proyecto

Output From:
Description
Main Description

El dialogo es una reunion que se realiza para que el gerente general de la empresa de a conocer

  • El funcionamiento de la empresa, ya que es la persona que mas conoce la empresa y sabe como funciona
  • El equipo de trabajo con el que se va a ejecutar el proyecto
  • Los requisitos que se necesitan para el proyecto
Brief Outline

Toda la información que se diga en este dialogo o reunion debe ser escrita informalmente para luego llevarla a una revisión.

Key Considerations

El dialogo se da para tener una estructura clara del proyecto entre el equipo de trabajo que se creó, el lider del proyecto y el gerente general.
