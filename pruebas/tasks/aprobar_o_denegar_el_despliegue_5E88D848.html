Task: Aprobar o denegar el despliegue
El despliegue debe pasar por una serie de revisiones para que pueda ser aprobado por el lider del proyecto.
Purpose
El proposito de esta tarea es poder ver el plan de despliegue y verificar que si este todo en orden
Relationships
RolesPrimary Performer:
  • Lider de proyecto
Additional Performers:
InputsMandatory:
  • Informe de pruebas
    		Optional:

    Outputs
    • acta de inicio de despliegue
      Main Description

      La aprobacion o denegacion del despliegue es sutil para que el proyecto con su despliegue pueda funcionar.
