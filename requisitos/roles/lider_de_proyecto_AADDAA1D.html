Role: Lider de proyecto
El líder de proyecto se encarga de comprender el negocio y ponerlo en términos concretos que le faciliten la comprensión al arquitecto
Synonyms:
  • Director de proyecto
  • Gestor de proyecto
  • Coordinador de proyecto
  • Jefe de proyecto
Relationships
Primary Performs
  • Construir y presentar proyecto al arquitecto
  • Revisar Estructura
  • Validar requisitos
Modifies
  • Estructura de software revisada
  • pre-proyecto
    Main Description

    El líder de proyecto cumple tareas como:

    • Analizar negocio
    • Construir y presentar proyecto al arquitecto
    • Validar requisitos
    • Revisar Estructura
    Staffing
    Skills

    El líder de proyecto tiene algunas habilidades como:

    • Gran conocimiento en software empresarial.
    • Capacidad de síntesis de información.
    • Capacidad de buena explicación.
    • Un liderazgo implacable.
    • Competencia de observación.
    • Capacidad analítica.
    • Coordinación
    • Orientación a resultados
    • Adaptabilidad
    • Gestión de riesgos
    • Establecimiento de objetivos
    • Capacidad de motivación
    • Toma de decisiones
    Assignment Approaches

    El líder de proyecto tiene diferentes enfoques de asignación tales como:

    • Documentar la información necesaria sobre la empresa para el desarrollo del proyecto.
    • Liderar al equipo.
    • Ser de apoyo para el entendimiento del proyecto
    Synonyms
    • Director de proyecto
    • Gestor de proyecto
    • Coordinador de proyecto
    • Jefe de proyecto
    Key Considerations

    El líder de proyecto:

    • Debe manejar términos que todo el equipo pueda entender.
    • Es el de mayor conocimiento sobre la empresa dentro del equipo.
    • Ayuda al arquitecto a entender el negocio.
    • Es el que analiza y verifica la estructura plateada por el arquitecto.
