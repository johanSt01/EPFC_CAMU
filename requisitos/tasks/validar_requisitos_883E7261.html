Task: Validar requisitos
El líder de proyecto valida los requisitos
Purpose
validar que los requisitos cumplan con las necesidades de la empresa
Relationships
RolesPrimary Performer:
  • Lider de proyecto
Additional Performers:
InputsMandatory:
  • Estructura de software documentada
    		Optional:
    • None
    Outputs
    • Estructura de software revisada
      Main Description
      El líder analiza detalladamente los requisitos y crea observaciones o recomendaciones en caso de que las necesite, se verifica que los requisitos funcionales y no funcionales sean los adecuados para el software que se solicita
      Key Considerations
      Los requisitos es de las partes primordiales del proyecto, por ello se debe realizar una verificación y análisis correcto y detallado
